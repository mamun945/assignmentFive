1️⃣ What is the difference between var, let, and const?
Ans:
#var
1.function scoped
2.not block scoped
3.can be redeclared
4.can be reassigned
5.hoisted initialize with undefined

#let 
1.block scoped
2.can not be redeclared
3.can be reassigned
4.hoisted but not initialize

#const 
1.block scoped
2.can not be redeclared
3.can not be reassigned
4.hoisted but not initialize 
5.must be assigned and declared same time
